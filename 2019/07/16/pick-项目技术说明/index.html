pick 项目说明

发表于 2019-07-16 | 分类于 pick 项目

项目介绍

pick项目出身于 2019 年 360 高校产品大赛，获得华北区第一名。
为娱乐社交类型 APP

项目进展

项目地址：http://47.102.*.*/p/login
目前已组建 5 人团队，前后台、产品、设计、运营各一名，与一家公司进行合作中。
项目方向为融合区块链技术和比特币思想，先作为插件进入平台，进行币改。

基于springboot+hibernate后台开发，基于vue前端开发，实现前后端分离。

后端技术

技术 名称
Spring Boot 容器+MVC框架
Spring Security 认证和授权框架
Hibernate ORM框架
MySQL MySQL数据库
Druid 数据库连接池

前端技术

技术 名称
Vue 前端框架
Vue-router 路由框架
Vuex 全局状态管理框架
Element 前端UI框架
Axios 前端HTTP框架

后台功能

  • 后台用户登录后获取token
  • 话题分类管理：控制显示、排序、编辑删除分类
  • 查看参与的话题订单，花费的pick币
  • 查看账户的的流水
